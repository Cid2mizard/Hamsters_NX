# Hamsters_NX
A breeding of Hamsters game in text mode for the Nintendo Switch.



Controls :

  - Right & Left : +/- 1

  - Up & Down : +/- 10

  - R & L : +/- 100

  - A : Next step

  - Plus : Exit
